# AGENT SECURITY RULES

Règles de sécurité pour agents IA :

- Lire avant d'écrire.
- Proposer un plan avant modification.
- Limiter le nombre de fichiers modifiés.
- Ne jamais modifier les secrets.
- Ne jamais lancer une commande destructive sans validation.
- Ne jamais toucher à la production sans GO explicite.
- Demander validation avant installation de dépendances.
- Préférer les actions locales, testables, réversibles.
