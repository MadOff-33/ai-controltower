# NO PROD WITHOUT GO

Règle absolue : aucune action production sans GO explicite.

Interdit sans validation humaine :
- git push
- déploiement serveur
- modification base de données distante
- appel API production
- modification WooCommerce, Stripe, Hostinger, Zyro, Google Cloud ou tout service réel
- suppression massive de fichiers
- migration structurelle
