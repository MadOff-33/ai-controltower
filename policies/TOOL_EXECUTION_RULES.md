# TOOL EXECUTION RULES

Les scripts du dossier tools/ sont des outils contrôlés.

Règles :
- Un agent peut lire un script.
- L'exécution passe par validation humaine.
- Les scripts de lecture et audit sont privilégiés.
- Les scripts destructifs sont interdits en V1.
- Les sorties importantes doivent être copiées dans le contexte de revue.
