# BMAD Web / SEO / Content — Adaptation Aider / Ornith

## Objectif

Utiliser les éléments BMAD WDS issus de Gator pour les projets web, SEO, contenu, design et tunnel de conversion.

## Usage

À utiliser pour :
- audit site ;
- contenu SEO ;
- landing page ;
- stratégie blog ;
- cohérence message/offre ;
- conversion ;
- documentation marketing.

## Règles

- Ne pas inventer de données.
- Séparer analyse, recommandations et actions.
- Toujours distinguer lecture seule et modification.
- Toujours vérifier les URLs, tracking, sitemap, indexation et risques SEO quand pertinent.
- Ne jamais modifier un site live sans GO explicite.
